# Data


## Input data

This project uses the following input data:
- diabetes_root: a dummy sample data containing some train and test datasets for diabetes

These data are stored into the Azure Storage account: testdlb29924085829

At execution time they are installed (mount or download) thanks to the AIP datasets defined into the ml_config.json file.


## Output data

The output data are generated at execution time and uploaded into Azure Storage accounts thanks to the AIP upload feature defined into the ml_config.json file.

# Introduction


## Background

This is a sample project built based on AI Platform cookiecutter project template.

AI Platform provides the necessary assets (AIP CLI, AIP SDK, AIP WebApp, AIP documentation, ...) to be able to interact easily with it.


## AI Platform

The goal of AI Platform is to help Data-Scientists to bring machine learning workflows to production faster.

It allows to :

- Provide the same code base for all Data Factory projects
- Gain in productivity for data scientists and data engineers
- Build code with pre-validated patterns and best practices (security, regulations,etc.)

Therefore, it gives to the data-scientists the opportunity to focus solenly on developing business code for training and evaluating models.
